# 功能总览

笺间围绕“文件组织、沉浸写作、智能辅助、可靠保存”设计。信笺使用 `.letterpaper` 格式，正文、素材、排版、评注和 AI 记录会一起保存。

## 文件与组织

### 工作区、文件树与大纲

- 左侧栏可在文件树和正文大纲之间切换。
- 文件树支持展开文件夹、双击进入、打开信笺，以及拖动信笺到当前可见的文件夹。
- 文件和文件夹的创建、重命名、备份、移动与删除均使用应用内输入或确认弹窗。
- 大纲由正文的一至三级标题生成，点击条目可跳转到相应位置。

### 标签页、打开队列与阅读位置

- 多个信笺可以同时打开，并通过顶部标签切换。
- 每个标签会记录自己的阅读位置；切走再返回时恢复上次滚动位置。
- 标签达到容量后继续打开信笺，会按打开队列腾出位置。
- 关闭标签不会删除文件；未保存信笺会先进入保存确认。

### 保存、恢复与导出

- 已保存信笺可直接写回，也可另存为新的 `.letterpaper` 文件。
- 未命名信笺会保存为临时会话，应用重启后可以恢复。
- 导出弹窗支持 PDF 和分页图片，可选择目标位置并查看处理进度。
- 分页图片依据正文分页符拆分，并以连续编号的 PNG 文件输出。

## 写作编辑

### 正文、标题编号与目录

- 标题、署名、日期和正文均可直接编辑。
- 正文支持一至三级标题、段落、列表、引用、对齐和撤销/重做。
- 标题层级同时驱动左侧大纲和正文目录；标题编号既可由模板控制，也可对当前标题单独启用或取消。
- 目录会按层级区分字号、颜色和缩进，标题仍保持正常的光标命中与编辑体验。

### 选区格式与链接

- 选中文字后出现悬浮工具条，可设置加粗、斜体、下划线、字体颜色和水彩标注。
- 下划线提供实线、波浪线、虚线和双横线。
- 链接支持 `http`、`https` 与邮箱地址；已有链接可编辑或移除，移除链接不会删除显示文字。
- AI 问答模式中可把选区加入“已标记文字”，作为下一次提问的重点上下文。

### 文档评注

- 框选文字后可新建评注，正文显示评注高亮，页面侧边显示对应锚点。
- 点击锚点可查看、编辑或删除评注。
- 评注范围会随正文变化进行映射，并随 `.letterpaper` 保存和复制。
- 同一位置评注过密时会限制继续添加，避免锚点重叠失控。

### 图片、音视频、引用与分页

- 媒体菜单支持图片、音频、视频和链接；图片可以调整宽度并编辑标题。
- 音频文件上限为 20 MB，视频文件上限为 100 MB，素材会打包进信笺。
- 元素菜单支持引用块、分割线、分页符和表格。
- 模板可以控制图片标题和编号的显示；隐藏时标题文字仍会保留。

### 表格

- 顶部工具栏可插入默认表格。
- 光标进入表格后，上方出现表格工具条。
- 工具条支持在上/下插入行、在左/右插入列、删除当前行列或删除整表。

## AI 功能

### 供应商、模型与连接测试

- Gemini、DeepSeek 和 Codex CLI 是不可删除的内置供应商。
- 可创建多个自定义供应商，协议可选 OpenAI 兼容或 Anthropic 原生。
- 创建时填写唯一名称、协议和 Base URL，随后在详情页配置 API Key 与模型标识。
- 只有测试成功的模型才能设为默认并用于 AI 优化或 AI 问答。
- 自定义供应商可以删除，但默认供应商必须先切换；删除时密钥和模型配置会同步移除。
- 界面和公开配置只显示密钥遮罩或末四位，不返回完整 API Key。

### Codex CLI

- Codex CLI 复用本机安装和登录状态，不在笺间保存登录凭据。
- 配置页可检查安装、登录、CLI 版本和模型目录，并按模型保存推理强度。
- Codex 可用后可以设为默认，供 AI 优化和 AI 问答使用；浏览器预览仅展示配置，真实调用限桌面端。
- 每次调用使用独立临时会话、禁止审批并保持只读沙箱；允许读取的项目范围由当前功能决定。

### AI 优化

- AI 优化读取当前信笺并生成优化稿。
- 定稿线以上作为背景，线以下作为本次优化重点。
- 优化结果按信笺保存，可清空、重新生成或插入定稿线。

### AI 问答

- AI 问答用于围绕当前信笺审阅、改写、找逻辑漏洞或生成标题。
- 消息、输入草稿和已标记文字按信笺保存；切换信笺或模型不会混用记录。
- 使用 Codex 时可选择仅当前信笺、信笺所在目录、整个工作区或指定工作区子目录。
- Codex 默认附加当前信笺全部原图，也可切换为仅发送 `[图N.标题]`；目录范围和图片模式都按信笺保存。
- 子目录会验证真实路径和工作区包含关系，失效或越界时要求重新选择，不会静默扩大读取范围。

## 视图与效率

### 左右分屏

- 普通写作模式可把一个已打开信笺固定到右侧分屏。
- 左右信笺可并排对照，左侧文件栏仍可继续使用。
- 右分屏只允许一个，可从标签右键菜单或右侧关闭按钮取消。

### 系统模板与用户模板

- 系统模板只读，可预览并直接应用；切换模板不会改变正文、素材或评注。
- 可基于系统模板创建用户模板，编辑名称、封面简介、信纸背景、字体、字号、颜色、段落、标题编号和图片标题规则。
- 用户模板支持自定义分组、重命名、归类和删除；“我的模板”始终保留完整列表。
- 可指定新建信笺默认模板；默认模板被删除时会回退到上一项有效选择。

### 状态栏、缓存与更新

- 状态栏显示字数、段落、页数、图片、引用、自动保存时间和缓存大小。
- 缓存用于加速已打开信笺的编辑器结构切换，可手动清理。
- 更新入口位于右下角；发现更新时显示状态提示，检查失败或无更新后恢复为可点击状态。

## 帮助中心

- 顶部“帮助”按钮打开应用内帮助中心。
- 帮助中心按文件与组织、写作编辑、AI 功能、视图与效率四类展示当前功能、步骤、注意事项与界面截图。
- 打开或关闭帮助不会改变编辑器、AI、标签页或分屏状态。
